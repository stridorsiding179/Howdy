# Howdy!

Howdy! is a Windows desktop app for managing Roblox accounts with premium license support.

Download the latest version from the Releases section.

## Features

- Windows desktop app
- Roblox account manager
- Premium/license key support
- Portable ZIP release
- Built with Nuitka

## Download

Go to the Releases section and download the latest `Howdy-v1.0.0.zip`.

## Notice

Howdy! is a new unsigned Windows app, so some antivirus engines may show false-positive warnings. The file has been submitted to Microsoft for review.
