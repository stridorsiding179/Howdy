# Howdy!

Howdy! is a Windows desktop app for managing Roblox accounts with premium license support.

## Features

* Multi-instance Roblox launching
* Account grouping and organization
* Custom layout management
* Scheduled launch support
* Auto rejoin support
* Session status checking
* Session recovery tools
* Network profile support
* Account setup helper
* Portable Windows release
* And more!

## Download

Download the latest version from the [Releases page](https://github.com/ElieCh1/Howdy/releases).

## Notice

Howdy! is a new unsigned Windows app, so some antivirus engines may show false-positive warnings. The file has been submitted to Microsoft for review.
